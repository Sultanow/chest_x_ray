File Analysis

The File Analysis page is the core element of our application. You can upload X-ray images and view the classification results of our Deep Learning model.

How does it work?

When you click the (+)-button, a file upload dialog will pop up. Now you can select a local file from your device. Note that only file types 'jpg', 'jpeg' and 'png' are supported. After uploading the file it gets transferred to our backend where the classification takes place. If you don't have a chest X-ray image available, you can use the 'Demonstration'-button instead. The website will then sent two example images to the backend.

What happens with the uploaded data?

Since medical data is very confidential, none of the uploaded data gets stored, buffered, cached or the like. The moment you leave the File Analysis page all data gets dropped. This also means that when you navigate through the application or refresh the page, all classification results are lost.
