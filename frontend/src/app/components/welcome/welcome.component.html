Chest X-ray UI

Deep Learning based pneumonia detection

About

With this tool you can analyze chest x-rays and detect viral and bacterial pneumonia within seconds.

Sequential three stage approach for lung segmentation, pneumonia detection and viral versus bacterial pneumonia distinction.

Open Source

The source code of this website and the underlying deep learning models are publicly available on GitHub.

Feel free to reach out to us if you are interested in the project.

Disclaimer

This tool is experimental and must not be used for clinical or commercial purposes.

Both the website and neural networks are under development. Functionality, layout and availability are subject to change.
